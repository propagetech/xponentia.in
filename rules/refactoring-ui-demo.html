Line Length

This paragraph demonstrates optimal line length. It's constrained to approximately 65 characters per line (45-75 range) for the best reading experience. Notice how comfortable it is to read compared to text that spans the full width of the screen.

Line Height is Proportional

Smaller text needs more line height (1.5) to help your eyes find the next line.

Large headlines can use tighter line height (1.1) since your eyes don't need as much help.
